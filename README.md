# Ritornello

**Canvas infinito de loop engineering** — orquestre agentes de IA em terminais sobre um canvas: desenhe ciclos com avaliadores, condições de saída (𝄋 segno) e budgets, componha loops em cascata e mantenha o veredito final sempre humano, no Pódio.

Este repositório hospeda os **downloads e o feed de atualizações** do Ritornello para macOS.

## Instalar

1. Baixe o `Ritornello-x.y.z.dmg` mais recente em [Releases](https://github.com/allansteve/ritornello-releases/releases/latest).
2. Abra o DMG e arraste o **Ritornello** para **Aplicativos**.
3. Na primeira abertura, o macOS pode avisar que o desenvolvedor não foi verificado: clique com o botão direito no app → **Abrir**, ou vá em Ajustes → Privacidade e Segurança → **Abrir mesmo assim**. (Só acontece uma vez.)

**Requisitos:** macOS 14 ou superior — Intel e Apple Silicon.

## Atualizações

O app verifica atualizações sozinho: quando houver versão nova, ele avisa e instala com um clique (menu **Ritornello → Verificar atualizações…** para checar manualmente). Cada atualização é assinada e verificada criptograficamente.

## O que vem dentro

- Terminais reais como nós de um canvas infinito
- Loops de agentes desenháveis: episódio → avaliador → ritorno, com compassos inspecionáveis
- `rito` — CLI embutido: agentes conversam entre nós (`rito say`), editam notas (`rito note`) e testam a própria UI no browser do canvas (`rito portal`)
- Pódio: custo por ciclo e veredito ship/block humano
